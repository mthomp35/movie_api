About the Movie API

Objective

To build the server­side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description Request Method URL Endpoint Query Parameters Request Body Format Response Body Format
Return list of all movies GET /movies None None A JSON object with a list of movies
Return data about a single movie by title GET /movies/:title Movie Title None A JSON object with data about a specific movie
Return data about a genre by name/title GET /movies/genres/:title Movie Title None A JSON object with data about the genre of a movie
Return data about a director by name GET /movies/directors/:name Director's Name None A JSON object with data about a director
Return list of all users GET /users None None A JSON object with data about all registered users
Allow new users to register POST /users None A JSON object with data about the user registering/being added A JSON object with data about the user that was registered/added
Allow users to update their user info based on username PUT /users/:username Username None A text message confirming successful update of user information
Allow users to add a movie to their list of favorites PUT /users/:username/favorites/:title Username, Movie Title None A text message confirming the selected movie was added to favorites
Allow users to remove a movie to their list of favorites DELETE /users/:username/favorites/:title Username, Movie Title None A text message confirming the selected movie was removed from favorites
Allow existing users to deregister DELETE /users/:username Username None A text message confirming successful removal of user registration and information
